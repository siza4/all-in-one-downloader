# All-in-One Social Media Downloader (Vercel)

This project contains a simple frontend (static) and serverless API endpoints (Node) for resolving downloadable URLs from popular social platforms.

**Files**
- public/: static frontend (index.html, privacy, terms, contact)
- api/: serverless functions for each platform
- vercel.json: Vercel config
- package.json: deps (for reference)

Deploy:
1. Push this repo to GitHub.
2. Import to Vercel and deploy.
3. Add AdSense script to public/index.html after approval.

Legal: Only use for public/non-infringing content. See public/privacy-policy.html for template.
